## Install

```sh
pip install pyttsx3
pip install SpeechRecognition
pip install openai
pip install datetime
pip install json
```

<ul>

  <li>Go to <a href="https://beta.openai.com/account/api-keys" target="_black">OpenAi Api</a> and copy your apiId here.
python openaiProject.py
